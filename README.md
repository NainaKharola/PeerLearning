# Peer Learning & Mentor Matching Platform (PLMM)

A modern web application that connects learners with mentors, helping students improve their technical skills through personalized mentorship, mock interviews, study groups, and placement preparation.

## Features

* Secure Authentication
* Role-Based Access (Learner, Mentor, Admin)
* Personalized Dashboard
* Mentor Discovery & Recommendations
* Session Booking
* Mock Interviews
* Mock Tests
* Skill Analysis
* Progress Tracking
* Placement Preparation
* Study Groups
* Learning Resources
* Notifications
* Admin Analytics Dashboard

## Installation

Clone the repository

```bash
git clone https://github.com/NainaKharola/PeerLearning.git
```

Move into the project

```bash
cd PeerLearning/frontend
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```
